## Sobre o projeto:
Este é um projeto escolar desenvolvido no curso de Informática para Internet da escola técnica ETEC de Praia Grande - Extensão Maracanã pela equipe de TCC ProLitterae

A ideia por trás do CalenTec é facilitar para os alunos de ETEC's verem as datas importantes do ano em sua escola, pois existem muitos eventos que ocorrem nestas escolas, o que confunde muitos alunos, este projeto então serviria para que as datas sejam cadastradas e os alunos apenas precisem abrir e visualizar quais dias e horários vão conter que eventos.

## Instalando o projeto:
Você poderá instalar e testar o projeto de duas maneiras: gerando um apk ou carregando ele localmente, antes lembre-se de possuir as seguintes técnologias no seu computador:

* > [NodeJS 22.18 ou superior](https://nodejs.org/pt)
* > [NPM 10.9 ou superior](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
* > [Angular 20.2 ou superior](https://angular.dev/installation)
* > [Ionic 5.4.16 ou superior](https://ionicframework.com/docs/intro/cli)


### Gerando um apk
Para gerar um apk executável é necessário possuir o aplicativo Android Studio, clonar este repositório em sua máquina, abrir a pasta com o Android Studio e seguir o [passo a passo para geração de um apk Ionic](https://cursos.alura.com.br/forum/topico-gerar-apk-do-app-27662)

### Abrindo o projeto localmente.
Para Gerar o projeto localmente, você precisará apenas clonar este repositório, e então abrir a pasta criada com seu terminal (ou no terminal da sua IDE favorita) e rodar a seguinte sequência de códigos no terminal:

* > npm install
* > ionic serve

Pronto! Agora sua aplicação estará rodando localmente no localhost.

## Utilizando o sistema:
A ideia de utilização do projeto é simples. Basta criar uma conta e começar a utilizar e ver os calendários, porém existem 3 tipos de contas: as de funcionário, de aluno e de responsável.
* A conta de funcionário te dá a possibilidade de criar, editar e remover eventos nos calendários dos meses do ano.
* A conta de alunos e responsáveis apenas te dá a possibilidade de visualizar os eventos e horários de cada um, através do clique encima do horário que aparecer no calendário
